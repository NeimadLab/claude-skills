## What you're looking at

Each image above is an **SVG preview** of a PISA canvas — a reusable slide structure that captures intent, layout, and visual treatment. The gray background and role labels are preview-only; the generated PPTX uses your theme colors and your content.

**To use any canvas:** install the pack, tell Claude what you need, and it picks the right canvas automatically.

```
Install the corporate essentials pack.
Create a 6-slide Q3 performance review using the executive persona.
```
