# Claude Skills

Open-source expertise modules you can drop into any Claude Project.

## Available Skills

### PISA — Presentation Intelligence

Analyze, generate, retheme, and review professional PowerPoint decks. 8 personas, 61 templates, 3 themes, schema V2.1.

[Get started →](https://github.com/NeimadLab/claude-skills/tree/main/skills/pisa)

## How Skills Work

A Claude Skill is a SKILL.md file that encodes domain expertise: workflow detection, data contracts, tool orchestration, and quality rules. Drop it into a Claude Project — Claude reads it and becomes a specialist.

No plugins. No API keys. No infrastructure. Just one file.
